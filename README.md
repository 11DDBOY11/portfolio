# Portfolio Website (README)

## 📌 About

This is a **personal portfolio website** for **Darshan Dashyal**, showcasing skills, projects, certifications, and contact details.


## 🚀 Features

* Modern responsive design
* Sections: About, Skills, Projects, Certifications, Goals, Contact
* Smooth animations and scrolling
* Custom cursor effect
* Project links and live demo integration

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## ▶️ How to Run

1. Save the file as `index.html`
2. Open it in any browser

## 📂 Sections

* **Hero** → Introduction
* **About** → Education & background
* **Skills** → Technical skills
* **Projects** → Work showcase
* **Certifications** → Achievements
* **Contact** → Get in touch

## 💡 Note

* Fully front-end based (no backend)
* Can be hosted on GitHub Pages or any hosting platform
